웹사이트를 만든 이유

여기까지 오기 전에 저는 Python, Java 등에 대한 동영상 강의들을 잠깐씩 보았습니다. 그런데 다들 문법이 어떻게 이루어져있는지를 설명하더군요. 내가 이걸 배워서 할 수 있는게 무엇일지 궁금했지만, 언어들만 배워서는 그게 해결되지 않았습니다. 그래서 Youtube에서 '생활코딩' 채널을 보며 html 페이지를 먼저 만들면서 프로그래밍을 배워야겠다고 생각했습니다. 눈에 보이는 것을 해결하다보면 조금 더 재미가 있을 것 같아서요.

2021년 12월 21일 현재의 구상으로는, 앞으로 배우는 여러 기능과 언어들을 이 웹페이지 상에서 구현해보는 것입니다. 이 사이트를 누가 방문하기는 할지 모르겠지만, 혹시 게시판 같은 것이 생기면 어떤 기능이 있으면 좋을지 의견 주세요. 능력이 된다면 기능들을 찾아서 추가해보면서 공부해보겠습니다.
